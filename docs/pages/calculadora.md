# Módulo: Calculadora

## Referencia al Código
- **Archivo principal:** [`src/pages/CalculatorPage.tsx`](../../src/pages/CalculatorPage.tsx)

## Descripción
Página de entrada de la herramienta para realizar cálculos rápidos mediante los componentes dedicados de la aplicación sin perder contexto.

- Su utilidad se basa en ser un acceso ágil ("Saldame Calculator") independiente a los componentes de control de gastos, proveyendo lógicas orientadas a estimaciones numéricas cruzadas.
