# Módulo: Categorías

## Referencia al Código
- **Archivo principal:** [`src/pages/CategoriesPage.tsx`](../../src/pages/CategoriesPage.tsx)

## Descripción
Interfaz que oficia de gestión tipo CRUD para los rubros o categorías en las que se clasifican los ingresos y egresos.

## Funcionamiento
- Permite al usuario dar de alta etiquetas personalizadas para asociarlas posteriormente al procesar los Santander Excels, la subida de PDF o una carga manual.
- Las categorías dictaminan gran parte de cómo las "Estadísticas" agruparán el análisis de los gastos.
