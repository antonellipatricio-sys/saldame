# Módulo: Etiquetas (Tags)

## Referencia al Código
- **Archivo principal:** *(Por implementar)* Ej: `src/pages/TagsPage.tsx`

## Descripción
Interfaz para la gestión de "Etiquetas". Funciona de manera paralela o complementaria a las Categorías, permitiendo añadir metadatos o agrupaciones secundarias a los gastos ("tags").

## Funcionamiento
- Funciona como un atributo descriptivo para facilitar los filtros e inspecciones detalladas.
- Un gasto puede pertenecer a una sola "Categoría" principal pero tener múltiples "Etiquetas" para un control más fino.
